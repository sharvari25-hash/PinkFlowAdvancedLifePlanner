## ✨ Overview

**PinkFlow** is a beautifully designed, aesthetically pleasing task management and life planning application. Built with a focus on user experience and visual appeal, it combines powerful productivity tools with a calming, pink-themed interface to help you organize your life without the stress.

Whether you're a student, a professional, or just someone looking to get organized, PinkFlow provides the tools you need in a package you'll love to look at.

---

## 🚀 Key Features

### 📋 Smart Task Management
- **Create & Organize:** Easily add tasks with titles, descriptions, due dates, and priorities.
- **Categorization:** Tag tasks as *Personal*, *Work*, *Study*, *Health*, or *Finance*.
- **Color Coding:** Assign custom colors (Pink, Blue, Green, Purple) for visual distinction.

### 📊 Multiple Views
- **List View:** A clean, vertical list for scanning your day.
- **Kanban Board:** A drag-and-drop interface to manage workflow (Todo → In Progress → Done).

### 🧘‍♀️ Focus Mode
- **Pomodoro Timer:** Built-in focus timer to help you stay productive without burnout.
- **Distraction-Free:** A dedicated overlay to keep you zoned in on the task at hand.

### 📈 Analytics & Insights
- **Productivity Tracking:** Visual progress bars for weekly goals.
- **Task Counters:** Real-time updates on completed tasks and pending items.

### 🎨 Personalization
- **Theme Support:** Switch between a bright, vibrant **Light Mode** and a sleek, comfortable **Dark Mode**.
- **Responsive Design:** Works beautifully on desktop, tablet, and mobile.

## 💻 Installation & Usage

1.  **Clone the repository**
    ```bash
    git clone https://github.com/sharvari25-hash/PinkFlow.git
    ```

2.  **Navigate to the project directory**
    ```bash
    cd PinkFlow
    ```

3.  **Open the application**
    Simply open the `index.html` file in your preferred web browser. No backend server or complex build step is required!

    > **Tip:** for the best experience, use a modern browser like Chrome, Edge, or Firefox.
